## Description

This application will allow the user to use this command-line terminal to:
* add/view departments 
* add/view roles 
* add/view employees
* update employee role.

<br>

  ## Installation

  Run:
  <br>
  _npm install_
  <br>
  then Run:
  <br>
  _node app.js_
  <br>
  in the command line (terminal)
